# Product Brief

Driftline is a native macOS desktop file transfer client for developers, operators, and power users who need secure SFTP-first file movement without the clutter of older FTP tools.

## Goals

- Make common file transfer workflows fast and legible.
- Store secrets only in Keychain.
- Make host trust visible and explicit.
- Provide a polished dual-pane macOS experience.
- Keep the architecture ready for additional protocols.

## Non-Goals For The First Milestone

- Full parity with every FileZilla feature.
- Cloud sync.
- Team collaboration.
- Telemetry.
- App Store distribution.
- Implementing every future protocol immediately.

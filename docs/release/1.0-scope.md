# 1.0 Scope

Driftline 1.0 is a secure SFTP release, not a broad protocol release.

## Ships In 1.0

- Dual-pane local/remote SFTP browsing.
- System SSH as the default backend.
- Native Swift SFTP as opt-in unless real-server and accessibility QA justify changing that.
- Password, private-key, and SSH-agent workflows through the backend that safely supports each one.
- Keychain-backed credentials, strict host trust, delete/overwrite confirmations, and redacted diagnostics.
- Transfers, recursive folders, conflict handling, compare/sync preview, bookmarks, recents, CLI, MCP, widget, and release packaging already present in the app.

## Out Of 1.0

- FTP, FTPS, WebDAV, S3, SMB, remote editing, scheduled sync, automatic updates, crash reporting, telemetry, hardware keys, Touch ID unlock, and SSH-agent auth in the native Swift backend.
- Native Swift SFTP as the production default until broader real-server QA, SSH-agent strategy, and manual accessibility/security QA are complete.

## Freeze Rules

- No new storage format without a migration test.
- No new network surface without a security note.
- No new user-facing string without all four localizations.
- No new transfer backend before 1.0 unless it replaces a proven blocker.

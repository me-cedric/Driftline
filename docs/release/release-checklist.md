# Release Checklist

## Scope Freeze

- Confirm `README.md`, `ROADMAP.md`, and `CHANGELOG.md` describe the release honestly.
- Confirm FTP/FTPS remain marked unsupported unless implemented and tested.
- Confirm native Swift SFTP remains marked experimental until real file operations pass integration tests.

## Validation

```bash
swift test
swift build
./scripts/ui-smoke.sh
./scripts/package-dmg.sh
./scripts/release-check.sh
```

If Docker Desktop is running:

```bash
./scripts/integration-sftp-server.sh start
DRIFTLINE_INTEGRATION_SFTP=1 swift test
./scripts/integration-sftp-server.sh stop
```

## Signing And Notarization

```bash
DRIFTLINE_SIGN_IDENTITY="Developer ID Application: Example" ./scripts/package-dmg.sh
DRIFTLINE_NOTARY_PROFILE="driftline-notary" ./scripts/notarize.sh
```

Do not mark the release notarized until `notarytool` accepts the DMG and `stapler validate` succeeds.

## GitHub Release

- Tag `v0.1.0`.
- Run the `Release` workflow.
- Upload `Driftline.dmg` and `Driftline.dmg.sha256`.
- Paste release notes generated by `./scripts/release-notes.sh`.
- Confirm artifact checksum matches the Homebrew formula draft.

## Homebrew

- Replace `OWNER`.
- Replace `sha256`.
- Test formula locally:

```bash
brew install --build-from-source ./Formula/driftline.rb
driftline --version
```

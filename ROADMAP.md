# Roadmap

## Milestone 1: Secure SFTP MVP

- Real SFTP remote listing.
- Keychain password and passphrase flow.
- SSH private key selection.
- Host fingerprint first-use trust sheet.
- Upload/download queue with progress.
- Conflict confirmation.
- Saved servers and bookmarks persistence.

## Milestone 2: Production Transfer Workflows

- Recursive folder upload/download.
- Retry, cancel, clear, and history views.
- Transfer speed and remaining time estimates.
- Reconnect after dropped connection.
- Terminal launch via Terminal.app and iTerm2 detection.

## Milestone 3: Distribution

- Signed app bundle.
- DMG packaging.
- Notarization workflow.
- Homebrew tap formula.
- GitHub Releases with checksums.

## Future Ideas

- Status bar transfer monitor.
- Desktop widgets for stats and quick connect.
- Raycast extension.
- Spotlight and Shortcuts integration.
- Finder extension.
- Quick Look integration.
- Encrypted server profile import/export.
- iCloud sync.
- WebDAV, S3, SMB, SCP, and rsync.
- Remote editing, diff, folder sync, transfer rules.
- Bandwidth throttling, proxies, jump hosts, SSH agent, hardware keys, Touch ID unlock.
- Sparkle updates.
- Crash reporting and telemetry, opt-in only.
- Localization and accessibility audit.

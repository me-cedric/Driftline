# User Stories

- As a developer, I can save an SFTP server without storing its password in plain text.
- As a developer, I can create, edit, duplicate, and delete saved server profiles.
- As an operator, I can inspect a host fingerprint before trusting a server.
- As a power user, I can browse local and remote folders side by side.
- As a keyboard user, I can create tabs, reconnect, upload, download, and search through shortcuts.
- As a maintainer, I can add a protocol adapter without rewriting the UI.

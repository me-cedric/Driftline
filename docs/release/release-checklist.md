# Release Checklist

## Scope Freeze

- Confirm `README.md`, `ROADMAP.md`, and `CHANGELOG.md` describe the release honestly.
- Confirm FTP/FTPS remain marked unsupported unless implemented and tested.
- Confirm native Swift SFTP remains marked experimental until real file operations pass integration tests.

## Validation

```bash
swift test
swift build
./scripts/ui-smoke.sh
./scripts/package-dmg.sh
./scripts/release-check.sh
```

If Docker Desktop is running:

```bash
./scripts/integration-sftp-server.sh start
DRIFTLINE_INTEGRATION_SFTP=1 swift test
./scripts/integration-sftp-server.sh stop
```

## Signing And Notarization

### First-time setup (done once)

Set up the following [GitHub Actions secrets](https://github.com/me-cedric/Driftline/settings/secrets/actions):

| Secret | Description |
| --- | --- |
| `DRIFTLINE_SIGN_CERT_P12` | Base64-encoded Developer ID Application certificate (.p12) |
| `DRIFTLINE_SIGN_CERT_PASSWORD` | Password for the .p12 certificate file |
| `DRIFTLINE_SIGN_IDENTITY` | Certificate name, e.g. `Developer ID Application: Your Name (TEAMID)` |
| `DRIFTLINE_APPLE_ID` | Apple ID email used for notarization |
| `DRIFTLINE_APPLE_TEAM_ID` | Apple Developer Team ID |
| `DRIFTLINE_APPLE_APP_PASSWORD` | App-specific password for notarytool |

To generate the base64-encoded P12:
```bash
base64 -i DeveloperID.p12 | pbcopy
```

### CI signing (automated)

When triggering the `Release` workflow, set:
- **`sign`** → `true` to code sign the app bundle
- **`notarize`** → `true` to notarize and staple the DMG (requires `sign=true`)

Do not mark the release notarized until the CI run completes successfully.

## GitHub Release

- Tag the release version.
- Run the `Release` workflow with desired inputs.
- Confirm the draft release is created with `Driftline.dmg` and `Driftline.dmg.sha256`.
- Paste release notes generated by `./scripts/release-notes.sh`.
- Confirm artifact checksum matches the Homebrew formula draft.

## Homebrew

- Replace `OWNER`.
- Replace `sha256`.
- Test formula locally:

```bash
brew install --build-from-source ./Formula/driftline.rb
driftline --version
```
